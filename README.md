Para realizar las tareas encontraran en el directorio distintas carpetas las cuales dentro tienen distintos tipos de ejercicios.

Los mismos pueden tener una instrucción para desarrollar la tarea desde 0, como tambien puede que tenga alguna ayuda o codigo mal escrito que deberán corregir. 

Dentro de cada carpeta encontraran un archivo con la extensión js.

Para resolver los mismos, deberán crearse una rama con su nombre (solo por esta vez a los fines de realizar la tarea) y trabajar allí. Una vez finalizada la tarea pueden realizar los paso necesarios para almacenar sus cambios a nivel local y luego remoto en github.
